Développement avec Eclipse Tools

IBM® Eclipse Tools for Bluemix® fournit des outils légers dans Eclipse qui permettent le développement et l'intégration rapides d'applications dans des clouds IBM® Bluemix® ou Cloud Foundry.

  1. Si vous ne disposez pas encore d'Eclipse, installez Eclipse Neon for Java EE Developers (4.6.1).

  2. Cliquez en maintenant le bouton de votre souris enfoncé sur le bouton suivant pour le faire glisser et le déposer dans la barre d'outils Eclipse, puis suivez les invites d'installation d'IBM Eclipse Tools for Bluemix :

  3. Téléchargez votre code de démarrage et importez-le dans Eclipse en sélectionnant Fichier>Projets existants dans l'espace de travail>Fichier archive.

Pour des instructions détaillées concernant le déploiement d'applications, voir Déploiement d'applications avec IBM Eclipse Tools for Bluemix.
