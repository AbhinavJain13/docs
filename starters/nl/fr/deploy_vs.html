Développement avec Visual Studio

  1. Si vous ne disposez pas encore de Visual Studio, installez Visual Studio 2015 ou Visual Studio Code en suivant les instructions décrites ici.

  2. Téléchargez votre code de démarrage et procédez à son extraction dans un dossier sur votre disque dur.

  3. Ouvrez le projet dans Visual Studio

    • Si vous utilisez Visual Studio 2015, ouvrez le fichier de solution (fichier .sln) en accédant à Fichier > Ouvrir > Project/Solution.
    • Si vous utilisez Visual Studio Code, ouvrez le dossier qui contient le fichier de solution (fichier .sln) en accédant à Fichier > Ouvrir.

  4. Une fois que vous avez modifié votre code, vous pouvez le redéployer à l'aide des outils d'interface de ligne de commande CF ou Bluemix.

Pour obtenir des instructions détaillées concernant le déploiement d'applications, voir Déploiement d'applications à l'aide de l'interface de ligne de commande CF.
