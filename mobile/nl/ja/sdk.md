# SDK
{: #sdk}

最終更新日: 2016 年 10 月 14 日
{: .last-updated}

{{site.data.keyword.Bluemix}} Mobile Services SDK をアプリに追加するには、使用する SDK を選択し、依存関係マネージャーを構成して SDK をアプリにプルします。
