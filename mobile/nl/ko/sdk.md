# SDK
{: #sdk}

앱에 {{site.data.keyword.Bluemix}} 모바일 서비스 SDK를 추가하려면 사용할 SDK를 선택한 후 앱에 SDK를 가져오도록 종속 항목 관리자를 구성하십시오.
