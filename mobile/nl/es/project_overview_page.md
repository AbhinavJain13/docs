# Página Visión general del proyecto
{: #project_overview}

Puede ver y trabajar con un proyecto del panel de control de {{site.data.keyword.Bluemix}} Mobile seleccionando el proyecto en la página Proyectos; se abrirá la página Visión general del proyecto.
{: shortdesc}

La página Visión general del proyecto muestra un mosaico de cada servicio configurado o disponible para que se configure con el proyecto seleccionado. El mosaico muestra el tipo de servicio y un botón *acciones* con tres puntos alineados en vertical. Estos servicios pueden incluir {{site.data.keyword.mobilepushshort}} o Analytics. Los servicios compatibles dependen del tipo de proyecto y de los servicios disponibles en la región, de modo que no todos los servicios estarán disponibles para que se asocien a todos los proyectos.  

 Cuando un tipo de servicio aún no está asociado al proyecto, se muestra el botón **Crear** en el mosaico. Las opciones del botón de acciones sirven para crear una instancia del servicio o para añadir un servicio existente cuando el servicio aún no está asociado. Si se selecciona **Añadir existente**, se muestra una lista de instancias de servicio de dicho tipo que no están asociadas a otros proyectos. Los servicios solo se pueden asociar a un proyecto. 

Si el servicio ya está asociado a este proyecto, el nombre de la instancia del servicio asociado se muestra en el mosaico tras el tipo de servicio. Esto facilita la búsqueda de la instancia de servicio relacionada con el proyecto en el panel de control de {{site.data.keyword.Bluemix_notm}}. Las acciones que están disponibles en el botón de acciones son **Ver** y **Eliminar** cuando el servicio ya está asociado al proyecto. Cuando se elimina una instancia de servicio solo se elimina la asociación a este proyecto; no se suprime la instancia del servicio del panel de control de {{site.data.keyword.Bluemix_notm}}.

Seleccione **Obtener código** en el mosaico Compilación para descargar el código correspondiente al proyecto. Para obtener más información sobre cómo descargar el código, consulte [Obtener código](get_code.html).
